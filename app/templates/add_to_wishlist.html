How to Find the BGG ID

To add a game to your wishlist, you can use the BoardGameGeek ID (BGG ID). Follow these steps:

  1. Go to BoardGameGeek and search for your game.
  2. Open the game's page and check the URL. The BGG ID is the number that appears in the URL.

    For example, in this URL:

    https:​//boardgamegeek.com​/boardgame​/342942​/ark-nova

    The BGG ID is 342942.

If you cannot obtain the BGG ID, you can enter the game name instead. Providing the BGG ID is preferred for accurate game data.
